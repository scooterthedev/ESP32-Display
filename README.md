# ESP32-Display

https://oshwlab.com/scooter_y/esp32-with-screen

This project is a customizable screen display board powered by an ESP32, featuring USB-C connectivity and six programmable buttons. It's coded using Arduino, offering a versatile and fun experience! Currently, there's a minor issue with the board, but don't worry—it’s stable and won’t explode anytime soon!
